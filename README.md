# 创意工坊双创管理系统 java746

## 项目概述

创意工坊双创管理系统是一款基于前后端分离架构的管理系统。该系统集成了多种功能模块，旨在为工坊管理员、教师和学生提供便捷的双创项目管理和日常运营服务。

## 技术栈

- 前端：Vue
- 后端：Spring Boot
- 数据库：MySQL
- 语言：Java

## 功能模块

### 基础模块

- 用户管理
- 部门管理
- 角色管理
- 菜单管理
- 日志管理
- 数据字典管理
- 文件管理
- 图表展示

### 业务模块

- 讲座管理
- 消息管理
- 项目管理
- 公共空间
- 房间预约
- 团队管理

## 系统角色

系统基于角色的访问控制，为以下角色提供使用：

- 工坊管理员
- 教师
- 学生

权限可精确到按钮级别，支持自定义角色和权限分配。

## 运行环境

- 前端：现代浏览器
- 后端：Java 8 或更高版本，Spring Boot 2.x
- 数据库：MySQL 5.7 或更高版本

## 部署步骤

1. 准备后端运行环境（Java、MySQL）
2. 部署后端服务（Spring Boot 应用）
3. 部署前端应用（Vue 应用）
